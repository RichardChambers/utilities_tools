========================================================================
    CONSOLE APPLICATION : FileAccessLib Project Overview
========================================================================

AppWizard has created this FileAccessLib application for you.  

This file contains a summary of what you will find in each of the files that
make up your FileAccessLib application.


FileAccessLib.vcproj
    This is the main project file for VC++ projects generated using an Application Wizard. 
    It contains information about the version of Visual C++ that generated the file, and 
    information about the platforms, configurations, and project features selected with the
    Application Wizard.

FileAccessLib.cpp
    This is the main application source file.  This contains the test harness we use
    in testing the simple index file library.

FileLib.cpp
    This is the simple index file library source code file.  It contains the source
    code for the simple index file library.

FileLib.h
    This is the simple index file library header file.  It contains the data structs
    that the user of this library can see as well as the function prototypes for the
    functions that make up the library.

/////////////////////////////////////////////////////////////////////////////
Other standard files:

StdAfx.h, StdAfx.cpp
    These files are used to build a precompiled header (PCH) file
    named FileAccessLib.pch and a precompiled types file named StdAfx.obj.
